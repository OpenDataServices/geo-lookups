# Contributing to geo-lookups

Contributions to `geo-lookups` are very welcome. You can contribute by:

- [raising an issue](https://github.com/drkane/geo-lookups/issues) for any gaps or enhancements you've spotted in the data
- submitting a pull request to fix any errors or missing data 
- suggesting additional data that it would be useful to have in the repository (by opening an issue)
